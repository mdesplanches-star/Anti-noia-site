# Anti Noia Finance

Projeto Next.js pronto para publicação inicial.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm install
npm run build
npm run start
```

## Publicar na Vercel

1. Crie um repositório no GitHub.
2. Envie esta pasta para o repositório.
3. Importe o repositório na Vercel.
4. Clique em Deploy.

## O que já vem pronto

- Layout responsivo com visual de app
- Lançamentos com criar, editar e excluir
- Metas financeiras
- Controle de cartões
- Persistência local no navegador
- Manifesto PWA e service worker base

## Próximos upgrades recomendados

- Login com Supabase, Firebase ou Clerk
- Banco de dados em nuvem
- Fechamento mensal real por competência
- Gráficos históricos
- Sincronização entre dispositivos
